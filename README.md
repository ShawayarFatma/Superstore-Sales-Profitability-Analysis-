# Superstore-Sales-Profitability-Analysis

Analyzed the Superstore dataset using SQL and Excel to identify revenue drivers, profitability challenges, customer behavior, discount impact, and growth opportunities.
Tools: SQL (PostgreSQL), Excel, GitHub

Key Business Findings
📌 Furniture contributes 32% of sales but only 2.5% profit margin
📌 Discounts above 20–30% consistently reduce profitability
📌 50% of customers generate 80% of revenue
📌 Only 20% of customers generate 80% of profit
📌 Revenue is concentrated in several low-margin products
📌 September–December is the strongest sales period (seasonality)

Business Recommendations
1. Reduce excessive discounting
2. Improve Furniture pricing and margin strategy
3. Retain high-profit customers
4. Promote high-margin, underutilized products
5. Run campaigns during low-demand months
